1. Amazon.co.uk Keyword Search

Where to go

Browser la Amazon.co.uk open pannu → search bar la namma target keyword type pannu

Steps

  1. Amazon.co.uk site open pannu
  2. Namma product keyword ah search bar la type pannu (e.g. "wall light")
  3. Enter click pannu — search results page varum, andha keyword ku top ranking products ella um list aagum

Sample

Amazon.co.uk la "wall light" nu search pannom → 40+ products list aana search results page open aachu.

2. Xray / ASIN Grabber (browser extension)

Where to go

Same search results page mela irundhukittu → browser toolbar la irukra Helium 10 extension icon click pannu → ASIN Grabber run pannu

Steps

  1. Helium 10 extension icon ah click pannu
  2. ASIN Grabber tool automatic ah search results mela oru data table kaamikkum — ella ASINs um BSR, price, reviews kooda varum
  3. BSR (Best Seller Rank) column ah paaru — number low ah irundha, adhu nalla selling product
  4. Namma niche ku close ah irukra, nalla BSR rank irukra 3-5 ASINs ah select pannu
  5. Select panna ASINs ah note pannu illa export pannu, adhu next step (Cerebro) ku use aagum

Filters (ASIN Grabber la ivalo paaru)

  • BSR (Best Seller Rank) — main filter, low number = high selling, adha base pannu select pannu
  • Price — namma price range ku close ah irukra products mattum paaka
  • Reviews — established, trust irukra products ah confirm pannikka

Sample

"wall light" search results la irundhu BSR rank base pannitu B0UKASIN01 (BSR #120), B0UKASIN02 (BSR #340), B0UKASIN03 (BSR #510) nu 3 ASINs select pannom.

3. Cerebro — Keyword Pull

Where to go

Helium 10 dashboard → Tools → Keyword Research → Cerebro

Steps

  1. Cerebro open pannu
  2. Step 2 la select panna 3-5 ASINs ah "+ Add Competitor" la paste pannu
  3. Get Cerebro Data click pannu
  4. Indha ASINs ella um rank aagra keywords full ah oru table la varum

Sample

3 UK ASINs vechu Cerebro run pannom → "wall light", "led wall light indoor", "modern wall sconce" madhiri 180+ keywords varuthu.

Apply

4. Search Volume Filter — Keyword Select

Where to go

Same Cerebro results table — column header la irukra Search Volume mela click pannu

Steps

  1. Search Volume column ah click pannu — High to Low sort aagum
  2. Top ah irukra high-volume keywords ah paaru
  3. Relevant ah irukra keywords mattum checkbox tick pannu (irrelevant/brand terms skip pannu)
  4. Select panna keywords ah namma final list ku eduthukka ready

Filters

  • Search Volume — High to Low sort, top 20-30 keywords mattum focus pannu
  • Relevance — namma product ku direct ah match aagra keywords mattum select pannu

Sample

High to Low sort pannina "led wall light" (8,400/mo), "wall light indoor" (5,200/mo), "modern wall sconce" (3,100/mo) top ah vandhuchu — indha 20 keywords ah select pannom.

5. Add to List

Where to go

Cerebro results table la, select panna keywords rows mela irukra "Add to List" button click pannu

Steps

  1. Step 4 la select panna keywords ah tick pannitu vechirunthadhu confirm pannu
  2. "Add to List" click pannu
  3. Puthu list create pannu (e.g. "Wall Light — Master List") illa existing list select pannu
  4. Save pannu — indha list ah aprom Keyword Tracker la kooda link pannalam

Yen idha pannurom

Indha keywords ah oru centralized list la vechirundha, easy ah track pannikkalam, team kooda share pannikkalam, and edhachum keyword miss pogaama irukkum — idhu dhaan "Add to List" ku main reason.

Sample

20 select panna keywords ah "Wall Light — Master List" nu puthu list create pannitu add pannom — ippo ella keywords um oru place la track aagudhu.

6. ChatGPT — New Title / Bullets / Backend Keywords

Where to go

Cerebro la irundhu keyword file export pannu (CSV/copy) → ChatGPT open pannu

Steps

  1. Cerebro la select panna keywords ah copy pannu illa CSV ah export pannu
  2. Namma ASIN details ready pannu — current title, bullets, category, key features
  3. ChatGPT open pannitu, ASIN details um Cerebro keyword file um rendaiyum paste pannu
  4. Prompt kudu: "Indha keywords ah use panni, indha product ku oru optimized title, 5 bullet points, backend search terms create pannu"
  5. ChatGPT tharra output ah review pannu — irrelevant/false claims edhachum irundha edit pannu
  6. Final copy ah Listing Builder illa Scribbles la paste pannitu, ella priority keywords um cover aaguthaa nu confirm pannu

Sample

"Wall Light — Master List" (20 keywords) + namma current ASIN title/bullets ah ChatGPT ku kudutha, puthu title + 5 bullets + 15 backend keywords generate aachu — Scribbles la paste pannitu 18/20 keywords cover aaguthu nu confirm pannom.
